Sobre nós

Quem somos

Somos parte da equipe do Compet do Cefet-MG que busca levar projetos relevantes para agregar na comunidade Cefetiana e proporcionar experiências únicas para os estudantes.

O que é o projeto Codelife?

O projeto Codelife é um site de educação voltado para iniciantes em programação, que visa ensinar a seu público a base da programação web( Js, HTML e CSS ) de forma gamificada, onde os conteúdos são separados por níveis (ilhas de conhecimento).

Trabalho desenvolvido pelos membros do Compet
